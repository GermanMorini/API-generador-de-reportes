¿Como usar esta API?

Enviá a alguna de las siguientes direcciones una petición POST con los datos indicados

/api/doc1

Genera un pequeño comprobante

            {
                "comprobante_id":"57d078a6-9679-4f5b-b6ad-e8bb47aa4fe0",
                "total":"420.069"
            }

/api/doc2

Genera un comprobante de asistencia a un parcial
